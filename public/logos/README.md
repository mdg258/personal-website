# Company Logos

This directory should contain the following logos:

## Required Logos

### 1. Cisco Meraki (`cisco-meraki.svg` or `cisco-meraki.png`)
- **Source**: [Cisco Brand Center](https://www.cisco.com/c/en/us/about/brand-center.html) or [Meraki Website](https://meraki.cisco.com/)
- **Preferred Format**: SVG (white version for dark background)
- **Fallback**: High-res PNG (transparent background)

### 2. USAA (`usaa.svg` or `usaa.png`)
- **Source**: [USAA Press Kit](https://www.usaa.com/inet/wc/about-usaa-press-kit) or official website
- **Preferred Format**: SVG
- **Fallback**: PNG with transparent background

### 3. Meta (`meta.svg` or `meta.png`)
- **Source**: [Meta Brand Resource Center](https://about.meta.com/brand/resources/meta/logo/)
- **Preferred Format**: SVG (white version for dark background)
- **Fallback**: PNG

### 4. U.S. Army (`army.svg` or `army.png`)
- **Source**: [U.S. Army Brand Guidelines](https://www.army.mil/) - Public domain
- **Preferred Format**: SVG or PNG
- **Note**: Multiple logo variations available (star emblem, wordmark, etc.)

### 5. Cornell University (`cornell.svg` or `cornell.png`)
- **Source**: [Cornell Brand Identity](https://brand.cornell.edu/design-center/logos/)
- **Preferred Format**: SVG
- **Options**: Cornell Seal, Wordmark, or Insignia
- **Note**: May require permission for commercial use

## Logo Guidelines

- **Background**: All logos should work on dark backgrounds (#0a0e14)
- **Format**: SVG preferred for scalability, PNG as fallback
- **Color**: White or light colored versions
- **Size**: Minimum 200x200px for PNGs
- **License**: Ensure compliance with each company's brand guidelines

## Temporary Placeholders

Until official logos are obtained, text-based placeholders are used in the components.
