# Portfolio Website

Personal portfolio website built with Next.js, TypeScript, and Tailwind CSS.

**Live Site:** [https://www.michaeldaleglenn.com](https://www.michaeldaleglenn.com)

## Tech Stack

- Next.js 16
- TypeScript
- Tailwind CSS v4
- Framer Motion
- Lucide React

<br>

<hr>

<br>

## Development

```bash
npm install
npm run dev
```


## Build

```bash
npm run build
npm start
```
